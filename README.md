## Getting Started

## Prerequirements

Python >= 2.7, < 2.8

### Install Python packages

Required:

```
pip install virtualenv
```

Highly recommended:

```
pip install virtualenvwrapper
```

### Install Yeoman

```
npm install -g yo
```
